## 📖 Sobre o Projeto

O **PrimeBank** é um projeto desenvolvido como parte de um trabalho acadêmico para o curso de **Sistemas de Informação na UNIRIO**. O objetivo central foi consolidar conhecimentos básicos em **Vue 3**, focando especialmente em como a reatividade funciona e no uso de seus métodos principais.

A aplicação simula um ecossistema bancário onde o usuário pode gerenciar suas finanças de forma simples, fluida e dinâmica, funcionando como uma Single Page Application (SPA).

### ⚙️ Funcionalidades Principais

* **Design Banking:** Interface inspirada em sistemas bancários modernos e limpos.
* **Gestão de Fluxo:** Adição de transações categorizadas como **Receitas** ou **Despesas**.
* **Gráfico Evolutivo:** Visualização do saldo acumulado em tempo real através de um gráfico dinâmico (Chart.js).
* **Extrato Dinâmico:** Listagem automática de todas as movimentações realizadas.
* **Experiência SPA:** Navegação rápida e sem recarregamento de página.

---

## 🚀 Tecnologias Utilizadas

* **[Vue.js 3](https://vuejs.org/):** Framework principal. Exploração profunda da Composition API, reatividade (`ref`, `reactive`) e métodos de ciclo de vida.
* **[Tailwind CSS](https://tailwindcss.com/):** Estilização utilitária para garantir um design responsivo e elegante.
* **[Vue Router](https://router.vuejs.org/):** Gerenciamento de rotas para transformar a aplicação em uma SPA.
* **[Chart.js](https://www.chartjs.org/):** Implementação de gráficos para análise visual de dados financeiros.

## 🛠️ Como Executar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone [https://github.com/SEU_USUARIO/primebank-vue.git](https://github.com/SEU_USUARIO/primebank-vue.git)
    ```
2.  **Instale as dependências:**
    ```bash
    npm install
    ```
3.  **Inicie o servidor de desenvolvimento:**
    ```bash
    npm run dev
    ```
